# Концепции
- Таск-трекер для малых команд

Веб-приложение и мобильное приложение для управления задачами в небольших командах (до 10 человек). Включает создание задач, назначение исполнителей, установку сроков, базовые канбан-доски, комментарии и уведомления. Фокус на простоту и минималистичный интерфейс без избыточного функционала.
- Платформа для микро-обучения

Мобильное приложение для получения знаний короткими порциями (5-10 минут в день). Пользователи могут создавать и проходить курсы на различные темы (от IT до кулинарии). Включает систему прогресса, напоминания и краткие тесты для закрепления материала.
- Сервис для совместного планирования поездок

Онлайн-платформа, где группа друзей или коллег может совместно спланировать путешествие. Функционал: сбор предложений по датам и направлениям, голосование за активности, общий бюджет, "копилка" для сбора средств, карта маршрута и общий фотоальбом.
- Система мониторинга здоровья растений

Мобильное приложение с интеграцией умных датчиков влажности почвы, освещенности и температуры. Приложение собирает данные с датчиков, анализирует их и дает пользователю персональные рекомендации по уходу за конкретным видом растения (полив, подкормка, освещение), а также отправляет напоминания.

# Критерии
- Срок реализации

Оценка того, насколько быстро можно разработать и выпустить минимально жизнеспособный продукт (MVP). Характеризует будущую работу над проектом. Меньше — лучше.

- Потенциал монетизации

Оценка потенциального дохода проекта или легкости внедрения платной модели (подписка, разовые покупки, реклама). Характеризует продукт. Больше — лучше.

- Сложность технической реализации

Оценка необходимых технических expertise, сложности архитектуры, интеграций со сторонними сервисами и масштабируемости. Характеризует будущую работу над проектом. Меньше — лучше.

- Уровень вовлеченности пользователей

Оценка того, насколько часто пользователи будут возвращаться к продукту и как долго будут его использовать. Высокая вовлеченность способствует вирусному росту. Характеризует продукт. Больше — лучше.
